Question 1: What are some of the benefits of using a static site generator? What are some disadvantages?

Advantages:

- Creates a website from .md files. Performs a lot of the overhead for creating a website
- Quick and simple way of getting up a website
- Creating a webpage is simple allow us to put up content quickly

Disadvantages:

- Cannot use any services such a database (no dynamic content on our webpage)
- There is no server side so we cannot perform things like server-side rendering
